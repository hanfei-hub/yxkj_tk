## 当前系统形态

- 前端：PySide6 Windows 桌面程序。
- 后端：FastAPI + SQLAlchemy。
- 线上后端：`http://120.26.207.89:8000`。
- 当前主要数据源：FastMoss 日本区新品榜 API。
- 当前大模型用途：标题翻译、AI 选品对话、衍生品生成、1688 图片匹配。
- 当前 1688 货源形态：已按第三方 API 方式做适配层，等待正式 API 参数后接入。
